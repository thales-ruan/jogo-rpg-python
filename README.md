# jogo-rpg-python atualização
## Sobre
- Execute o arquivo batalha.py
- A função batalhar , recebe os dois lutadores como parametro obrigatorio
- Ja tem um exemplo de como funciona
- se passar "item_loja" no argumento da função batalhar() ao terminar o vencedor compra um item da loja que ja esta programado
- Na proxima batalha ele tem mais 10 de dano com o item comprado 
- Os ataque estao programado pra um atacar de cada vez
- voce pode visitar a Loja com a chamada listar item 
- A cada batalha ganha o vencedor ganha 60 ouros 
- Não é possivel comprar item sem o ouro sufucuente
